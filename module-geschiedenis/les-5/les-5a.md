**While**

1\) Welke van deze programma's print **3 keer "Hallo"** uit?

Schrijf de goede code over in je schrift!

```python
1:
punten = 1
while punten == 3:
  print("Hallo")
  punten = punten + 1

2:

while punten < 3:
  print("Hallo")
  punten = punten + 1

3:

  print("Hallo")

```
